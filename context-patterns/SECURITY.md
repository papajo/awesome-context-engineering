# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing security@context-patterns.example.com.

## Supported Versions

We actively maintain security updates for the latest release branch.

## Security Best Practices

- Keep dependencies updated.
- Use secure coding practices.
- Regularly audit code for vulnerabilities.
- Respond promptly to reports.

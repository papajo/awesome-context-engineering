# Courses

Courses provide structured, comprehensive learning paths for context engineering — covering everything from foundational LLM concepts to advanced agent architectures. This section curates the best available courses, organized by provider, level, and focus area.

## Learning Path

### From Beginner to Advanced

1. **Beginner Path (1-2 months)**
   - Hugging Face NLP Course (Chapters 1-6)
   - DeepLearning.AI: LangChain for LLM Application Development
   - Cohere: LLM University

2. **Intermediate Path (2-4 months)**
   - DeepLearning.AI: Building RAG Applications with LlamaIndex
   - Stanford: CS224N (lectures on attention and transformers)
   - DeepLearning.AI: Functions, Tools, and Agents with LangChain

3. **Advanced Path (4-6 months)**
   - DeepLearning.AI: Advanced RAG Patterns
   - Stanford: CS329P — Production ML
   - DeepLearning.AI: Multi-Agent Systems with AutoGen

# Code of Conduct

## Our Pledge

We pledge to foster an open, welcoming, inclusive, and respectful community for all contributors and users.

## Our Standards

- Be respectful and considerate.
- Assume positive intent.
- Use inclusive language.
- Do not harass or discriminate.
- Respect privacy and confidentiality.
- Follow repository guidelines and GitHub's community standards.

## Reporting

Please report any violations of this code to the repository maintainers.

## Enforcement

Maintainers will address issues fairly and promptly to maintain the community's health.
